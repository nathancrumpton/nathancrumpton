## Nathan Crumpton

Network Technology student at Swinburne University of Technology, based in 
New South Wales, Australia.

I spent fifteen years in heavy industry and technical trades before moving 
into technology. Conveyor systems, industrial diagnostics, vehicle 
refinishing. That work was about the same things security work is about: 
understanding how a system fails, catching it before it does, and documenting 
it properly.

Now I build web tools and study network infrastructure and cybersecurity.
